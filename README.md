Getting Start
---
Firtst, You run Graphpack Server.
access http://localhost:4000/graphql from the browser.
```
cd server
npm i
npm run dev
```
Second, You run Nuxt App.
```
cd client
npm i 
npm run dev
```
Finally, You can access http://localhost:3000 from the browser!
